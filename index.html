Panels & navigation

  • T — Text-In · Y — Text-In Auto · I — send message once
  • N — Top menu (radio) · P — Avatar / WebM settings · M — Mixer · G — Options diamond
  • Esc — closes panels in order; if none open, returns to start screen

Radio & visuals

  • V — random station · B — previous station · Space — play/pause Deck A
  • C — next visual preset (when supported) · , / . — previous / next visual mode

Avatar (WebM) overlay

  • W — show/hide overlay · S — reset layout · A / D — previous / next clip
  • Q / E — playback speed · Z / X — opacity · − / = — scale
  • Arrow keys — move position

DJ Deck B

  • H — QUEUE · J — VIDEO · K — KARAOKE (same as buttons on Deck B)

Shortcuts are ignored while typing in fields, or when Lock UI is on (except L toggles lock). Open Text-In anytime from the center Options (🪭) triangle.
